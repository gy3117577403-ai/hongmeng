# v1.13.10 布局层级与预览区优化验收记录

版本：v1.13.10-layout-layer-preview-polish

状态：本地开发验证，等待统一部署验收。本文档不包含数据库密码、对象存储密钥、SESSION_SECRET、账号密码或任何真实密钥。

## 修复范围

- 顶部资料库菜单、管理员菜单、工单更多菜单改为 Portal 浮层，挂载到 `document.body`。
- 连接器参数表格行内“更多”菜单改为 Portal 浮层，避免被表格滚动容器裁切。
- 统一全站关键层级：页面内容、右侧工具栏、顶部导航、下拉菜单、弹窗、toast。
- 图纸资料库详情页移除大块重复信息卡，压缩为“规格 + 客户 / 品名 / 资料状态 / 更新时间”的紧凑标题区。
- 图纸资料库详情页调整为分类栏、主预览区、文件列表三栏，主预览区优先。
- 图纸资料库右侧文件列表保留文件名、类型、版本、大小、下载和删除，文件名允许两行显示。
- 当前分类无文件时只保留一处空态说明。

## 层级规则

- 普通页面内容：低层级。
- 右侧工具栏 / 抽屉：低于顶部导航和菜单。
- 顶部导航：高于页面内容与工具栏。
- Dropdown / Menu / Popover：高于顶部导航和工具栏。
- Modal / 全屏预览：高于 dropdown。
- Toast：最高提示层。

## 图纸资料库详情页重排

详情页顶部不再展示客户、客户编码、产品规格、品名、完整度、最近更新、备注等多张信息卡。

新的标题区：

- 第一行：规格。
- 第二行：客户、品名、资料完整度、缺失状态、最近更新时间。
- 右侧：上传文件、编辑、删除。

周计划字段不会显示在图纸资料库详情页。

## 右侧工具栏层级

生产工单页面右侧工具栏仍保持“信息 / 上传 / 操作 / 队列”的窄栏和覆盖式面板。

工具栏层级低于顶部菜单，因此不会遮挡：

- 管理员菜单
- 资料库菜单
- 工单更多菜单
- 全局搜索结果

## 预览区比例优化

生产工单和图纸资料库均以 PDF / 图片预览为核心区域。

图纸资料库三栏比例：

- 左侧分类栏保持紧凑，并完整显示分类名称。
- 中间预览区占最大面积。
- 右侧文件列表保持可读但不过度挤压预览区。

## 平板横屏验收

重点适配：

- 1024x768
- 1280x800
- 1366x1024

验收要点：

- 顶部按钮不换行。
- 菜单不被预览区、PDF 工具栏或右侧工具栏遮挡。
- 图纸资料库详情标题不再占用大块空间。
- PDF / 图片预览区明显扩大。
- 当前分类无文件时不出现重复空态。
- 文件列表中文文件名可读。
