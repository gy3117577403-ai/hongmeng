# QA：图纸资料库工作台重排与异常规格治理

版本：v1.14.5-drawing-library-workspace-redesign

## 验收范围

- 左侧客户列表和规格列表合并为紧凑浏览器。
- 详情标题区压缩，不再重复显示客户、规格、品名和完整度大卡片。
- 中间预览区放大，分类栏位于预览区左侧。
- 右侧文件列表只显示当前分类文件，并集中下载 / 删除操作。
- 空态去重，避免同屏多处显示暂无文件。
- 异常规格默认隐藏，并通过“异常数据”筛选集中处理。
- 批量导入原图拦截日期型规格。
- 平板横屏优先适配 1366x1024、1280x800、1024x768。

## 异常规格规则

以下规格会被视为异常：

- 空字符串。
- 纯日期，例如 `2026-03-05`、`2026/03/05`、`2026.03.05`。
- 日期时间，例如 `2026-03-05 10.17`、`2026-03-05 10:17`。
- 纯中文通用文件名，例如“图纸”“扫描件”“最终版”。
- 过短且缺少字母数字组合的文本。
- 缺少有效字母或数字编号的文本。

异常记录不会被自动删除。进入“异常数据”筛选后可以查看并编辑修正。

## 批量导入拦截

网页端和命令行导入共用同一套规格校验：

- 日期型文件名进入未匹配。
- 不自动创建日期规格的 DrawingLibraryItem。
- 不上传未匹配文件。
- 正式上传仍需要确认码。
