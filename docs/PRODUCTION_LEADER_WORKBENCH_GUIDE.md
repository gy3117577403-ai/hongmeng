# 生产组长工作台使用指南

版本：v1.17.0-production-leader-workbench
状态：本地验证候选，尚未部署 Sealos

## 页面定位

生产执行中心位于 `/production`，面向产线组长快速查看当前启用周的数量进度、交期风险和需要人工确认的事项。登录后仍使用共享业务数据，不新增角色权限。

## 四阶段卡片

- 未发图：显示规格、优先级、客户与品名、目标数量、交期及图纸异常；正常配料和正常资料状态不占卡片空间。
- 在前端 / 在后端：显示规格、客户与品名、累计完成数、目标数、百分比、进度条、剩余量及最多两个异常。
- 已完成：使用 64-82px 紧凑卡片，只显示规格和数量结果；尾数未清、超产会单独提示。
- 点击规格直接进入对应工单的原图资料预览，不再提供重复的“打开资料”按钮。

1024px 宽度下已完成列默认折叠。点击列头可展开或收起，状态仅保存在当前浏览器会话中。

## 数量口径

目标数量来自 `uncompletedQty`，累计完成数量来自 `completedQty`。页面统一计算剩余量、超产量和百分比：

- `3000 / 2990`：99.7%，剩余 10 套。
- `3000 / 3000`：100%，完成。
- `3000 / 3050`：101.7%，超产 50 套。
- 已完成但仍有剩余：标记“尾数未清”。
- 目标为空或数量无效：不猜测数量，显示待补充状态。

## 异常优先

卡片不显示“图纸已发”“配料已齐”“资料完整”等正常标签。仅显示需要处理的状态：图纸待发、待样品确认、待客户确认、图纸需变更、配料未齐、逾期、尾数未清、返工和规格异常。每张卡片最多展示两个异常，其余可在详情中查看。

顶部摘要提供今日交期、已逾期、图纸待确认、配料异常、尾数未清、紧急和已完成统计。摘要与快捷筛选均可点击，多个条件按组合筛选执行；“全部”清空快捷条件。资料完整度继续保留在高级筛选中。

## 快速确认

未发图卡片的“更新图纸状态”菜单支持：未发、已发、待样品确认、待客户确认、图纸需变更和已确认。更新成功后只更新当前卡片与摘要，并写入操作日志和进度日志，不整页刷新。

在前端 / 在后端卡片可通过“更新状态”推进阶段。标记为已完成前会展示数量摘要；仍有剩余量时明确提示将成为“尾数未清”，必须再次确认。已完成卡片的撤回操作收纳在更多状态菜单中。

## 快速更新进度

点击进度区域或“更新进度”会打开右侧覆盖式侧栏。侧栏只要求填写累计完成数量和进度备注，并显示当前目标、完成与剩余。保存后追加进度日志并局部更新卡片；达到目标时只提示是否同步完成，不强制改变状态。

## 排序与返回

进行中列按紧急、逾期、今日交期、图纸待确认、计划时间、剩余量和规格排序；已完成列按完成时间倒序。排序在共享服务端逻辑中执行。

从规格进入工单资料前，页面会保存搜索、筛选、横向滚动和各列纵向滚动位置。通过“返回生产执行”或浏览器后退返回时恢复原位置，恢复后清理一次性会话状态。

## 数据边界

本版本没有 Prisma schema 变化或 migration。工单、进度日志、生产资料、图纸资料库、连接器参数和 S3 对象均不删除；连接器说明书与 Windows 微盘导入助手不在本次修改范围内。
