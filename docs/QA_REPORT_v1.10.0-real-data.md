## 真实数据问题与修复

验收发现真实 Excel 中存在模板字段外的非空列，例如“波纹管”和“直径”。v1.10.0-rc.2 原逻辑只读取模板字段，会导致这类真实资料在导入时丢失。

本次已做最小修复：

- 不新增数据库字段。
- 不改页面业务范围。
- 对未映射但有表头的非空列，自动追加到备注字段。
- 示例格式：`波纹管：70mm带屏蔽1500v；直径：25`。

修复后再次预览和导入通过，额外列内容可在备注中查询和导出。
