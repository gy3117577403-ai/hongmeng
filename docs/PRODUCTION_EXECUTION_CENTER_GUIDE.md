# 生产执行中心使用指南

版本：v1.17.0-production-leader-workbench
状态：本地验证候选，尚未部署 Sealos

## 1. 页面定位

生产执行中心位于 `/production`，只展示当前启用周的生产工单。顶部入口可前往生产工单、周计划、图纸资料库、连接器参数和组装说明书，各模块数据边界不变。

## 2. 三种视图

- 生产看板：未发图、在前端、在后端、已完成四列，每列独立纵向滚动。
- 今日任务：今日交期、逾期、紧急、今日更新和今日完成的现场任务。
- 异常任务：缺图纸、配料未齐、资料不完整、交期缺失、规格异常和客户缺失。

“今日任务”只保留为视图，不在生产看板快捷筛选中重复出现。

## 3. 摘要与筛选

摘要统计始终表示当前启用周全量数据；工具栏右侧单独显示当前过滤结果数量。今日交期、已逾期、图纸待确认、配料异常、尾数未清、紧急和已完成摘要均可点击筛选，再次点击可取消。资料不完整移动到高级筛选，不再占用常驻摘要。

高级筛选支持：

- 当前周客户搜索与多选。
- 今日、明日、逾期、本周和自定义交期范围。
- 状态、优先级、图纸状态、配料状态和资料完整度。
- 已应用条件以筛选标签显示，可逐项移除或全部清空。

搜索使用 300ms 防抖，旧请求会取消；视图、搜索、筛选和当前周同步到 URL。CSV 导出复用相同服务端参数解析，导出结果与页面筛选一致。

## 4. 工单卡片

卡片以规格为主标题，不以内部 WO 编号作为主显示，并按阶段使用不同信息结构：未发图聚焦目标、交期与图纸异常；在前端 / 在后端聚焦完成数量、目标、百分比、剩余量和异常；已完成只显示规格与数量结果。

- 点击规格：进入该工单现有 Dashboard 原图资料预览，不再显示重复的“打开资料”按钮。
- 更新进度：右侧覆盖式侧栏只填写累计完成数量和进度备注。
- 图纸状态：未发图卡片可快速选择未发、已发、待样品确认、待客户确认、图纸需变更或已确认。
- 状态：打开紧凑快捷菜单；完成时展示数量摘要并对尾数未清二次确认。
- 详情：查看生产、工单资料、进度历史和来源字段。
- 查看图纸资料库：详情中的独立入口，不替代工单资料预览。

负责人和工位已从卡片、筛选、更新弹窗、批量操作及 CSV 中停用。数据库兼容字段和历史进度日志不删除、不篡改。

正常图纸、配料和资料完整状态不显示标签。异常统一由 `getProductionAlerts` 生成，每张卡最多显示两个；数量由 `getProductionQuantitySummary` 统一解析和计算。

## 5. 批量操作

默认不显示复选框。点击“批量操作”后才显示选择框和批量操作栏，可批量设置优先级、状态或追加进度备注。退出批量模式会清空选择；批量完成仍需输入 `COMPLETE_BATCH`。

## 6. 资料预览与返回

进入资料预览前会生成一次性 `returnKey`：

- URL 保存视图、搜索、筛选、周和页码。
- `sessionStorage` 仅保存看板横向位置与四列纵向位置，不保存密码、令牌或业务文件内容。
- Dashboard 显示“返回生产执行”。按钮与浏览器原生后退均可恢复原状态。
- 返回状态 30 分钟过期，成功恢复后自动清理。
- 登录跳转使用安全的站内 `next` 参数，不丢失 Dashboard 的 `from` 与 `returnKey`。

## 7. 菜单交互

管理员、资料库、状态、更多和高级筛选均使用统一 Portal 菜单：点击外部关闭、Escape 关闭、同一时刻只保留一个普通菜单、页面失焦或路由变化时关闭。Escape 关闭后焦点返回触发按钮；普通菜单不会锁定看板滚动。

## 8. 滚动与平板操作

- 页面根节点固定为 `100dvh`，不使用 body 承担工单滚动。
- 看板外壳只负责横向滚动，1024px 下可左右滑动。
- 四个状态列表分别负责纵向滚动，支持鼠标滚轮和单指触控。
- 列标题固定，列边界使用 `overscroll-behavior` 抑制页面跳动。
- 玻璃模糊只用于导航、统计、工具栏、列容器和浮层；卡片不使用模糊。
- Android WebView 标识存在时自动改用高不透明度背景并关闭大面积模糊。

## 9. 状态更新

状态、图纸状态与进度更新先在当前卡片做乐观显示，服务端失败会回滚并显示错误；成功后以服务端结果替换卡片并后台同步摘要。不会整页刷新。进度日志继续只追加，不覆盖历史记录。

已完成列采用紧凑卡片并支持折叠；1024px 宽度下默认折叠，用户选择写入 `sessionStorage`。完成但剩余数量大于 0 时显示“尾数未清”，完成数量超过目标时显示超产量。

## 10. 无数据与故障

- 无启用周：提示前往周计划中心启用。
- 空列：显示紧凑“当前状态暂无工单”。
- 首次加载：显示骨架卡片。
- 请求失败：保留当前 URL 筛选并提供“重新加载”。
