## 变更范围
- `WorkOrder.customerName` 可选字段，使用 Prisma migration 追加列，不删除或重写已有数据。
- 工单新增 / 编辑表单支持客户名称。
- 工单卡片、顶部当前工单信息条、打印摘要、工单直达链接和二维码展示客户名称。
- 工单搜索、CSV 导入、CSV 导出、元数据 JSON 导出和全局搜索同步支持客户名称。
- 首页顶部当前工单信息重排为不超过 64px 的常驻窄条，文件标题区只显示文件信息。
- 右侧资料信息 / 上传 / 操作 / 队列区域收口为可折叠、可调整宽度的工具窗，状态保存在浏览器 localStorage。
- 底部文件缩略图改为透明悬浮条，支持展开 / 收起，不参与预览区高度计算。
- 顶部操作按钮保留下载全部、刷新和更多操作，减少平板横屏拥挤。
