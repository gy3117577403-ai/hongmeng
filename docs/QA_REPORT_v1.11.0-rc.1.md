## 空态引导验收

- 当前分类无文件时显示“当前分类暂无文件”引导卡。
- 引导卡展示当前分类、支持格式 PDF/JPG/PNG、生产资料建议。
- 当前工单缺少必填资料时展示缺失分类。
- 空态下“上传 PDF”“上传图片”“拍照上传”按钮复用现有上传和拍照逻辑。
- 空态卡为白色/浅橙风格，不再只显示大块灰色空白。

## 缩略图隐藏逻辑验收

- 当前分类文件数为 0 时，不显示底部“文件 0 个”入口。
- 当前分类文件数为 0 时，不显示缩略图浮层。
- 当前分类有文件时，保留原有“文件 N 个”入口和透明缩略图浮层逻辑。

## 右侧工具栏空态感知验收

- 当前分类无文件时，右侧工具栏默认高亮“上传”。
- “信息”页显示当前分类、文件状态“暂无文件”和支持格式。
- “操作”页在无当前文件时禁用下载当前、编辑文件和删除文件。
- 当前工单没有任何文件时，“下载全部”禁用并显示明确提示。
- “队列”保持可用，上传按钮始终可用。

## 分类栏验收

- 分类栏宽度保持在平板可读范围。
- 分类名称完整可读，不出现 `S...` 或文字截断。
- SOP指导书、辅料规格、注意事项等分类支持两行展示。
- 每项保留图标/简称、文件数量、状态点和选中态。

## 当前工单条资料状态验收

- 不再显示“无资料”。
- 所有分类都没有文件时显示“资料 0/5”。
- 必填分类缺失时显示“缺资料 N”。
- 必填分类齐全时显示“资料完整”。
- 资料状态标签 tooltip 显示缺失分类。
- 不影响工单状态和优先级快捷修改。
