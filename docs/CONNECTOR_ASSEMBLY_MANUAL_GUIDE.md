# 连接器组装说明书资料库使用指南

版本：v1.16.1-connector-manual-bulk-import

状态：本地开发与自动化验证候选，尚未部署 Sealos。

## 模块定位

连接器组装说明书资料库用于长期保存连接器装配类 PDF 或多图文档，重点管理版本、章节目录、适用型号和全文检索。它与连接器参数资料互相独立：

- 连接器参数保存型号、外剥皮、内剥皮、入长和备注等结构化参数。
- 组装说明书保存 PDF / 图片、制造商、系列、版本、目录和说明文字。
- 一份说明书可以关联多个连接器参数，一条连接器参数也可以关联多份说明书。
- 解除关联不会删除参数、说明书或对象存储文件。

所有登录账号共享同一套数据。当前系统不做角色权限区分，所有写操作都会写入操作日志。

## 进入方式

可通过以下入口进入 `/connector-assembly-manuals`：

1. 顶部“资料库”菜单中的“连接器组装说明书”。
2. 连接器资料库模块切换中的“组装说明书”。
3. 连接器参数表中有说明书关联时，点击“说明书 N”。
4. 顶部全局搜索的“连接器组装说明书”或“说明书文件”结果。

连接器资料库模块切换保留四个分支：连接器参数、组装说明书、原始资料附件、导入批次。

## 工作台布局

页面采用平板横屏三栏布局：

- 左侧：说明书列表、制造商 / 系列 / 适用型号筛选和分页。
- 中间：当前版本 PDF 或图片预览。
- 右侧：说明书信息、章节目录、版本历史和关联型号。

搜索输入采用 300ms 防抖，并会取消旧请求。列表查询不会加载 PDF 或图片文件内容，只有选中文件后才访问受认证文件流 API。

## 新增说明书

页面默认优先显示“批量导入说明书”，用于文件夹或多文件零表单导入；“单份新增”保留为次级入口。批量流程包含本地轻量解析、只读预览、SHA-256 去重、版本建议、强确认、并发队列、失败重试和批次结果，详见 [连接器组装说明书批量导入指南](CONNECTOR_ASSEMBLY_MANUAL_BULK_IMPORT_GUIDE.md)。

批量导入不会要求第一次就补齐所有字段。文件名去掉扩展名始终作为默认说明书名称，自动检测标题单独保存，不强制覆盖默认名称。

### 单份新增

点击“单份新增”，填写：

- 说明书名称，必填。
- 制造商。
- 连接器系列。
- 文档编号。
- 关键词。
- 摘要。
- 首版版本号，必填。
- 发布日期。
- 文件类型：PDF 或图片集。

单份表单支持直接选择 PDF 或图片。PDF 会先读取前两页生成建议，点击“使用识别结果”后才应用建议字段；保存时可创建说明书并立即上传，也可以只创建基本信息、后补文件。

## 上传 PDF

PDF 版本每个版本保存一个有效 PDF。默认限制为 100MB。上传后系统会：

1. 把文件写入 S3 兼容对象存储。
2. 把元数据写入 PostgreSQL。
3. 读取 PDF 页数。
4. 尽可能提取 PDF 文字到版本全文索引。
5. 保留扫描型 PDF；没有可提取文字时仍可按标题、型号和关键词搜索。

服务端 PDF.js worker 使用已安装的 `pdfjs-dist` 文件，兼容 Next.js standalone 容器。不会新建第二套 PDF 渲染器。

## 上传图片集

图片集支持 JPG、JPEG、PNG、WEBP：

- 单张默认不超过 20MB。
- 一个版本最多 50 张。
- 图片按 `sortOrder` 排列。
- 在“版本”面板中可用上移 / 下移调整顺序。
- 预览时只挂载当前图片，不一次性解码全部大图。

## 上传新版本

点击“上传新版本”，填写版本号、发布日期、文件类型、状态、备注和章节目录。

- 同一说明书内版本号不能重复。
- 新版本默认成为最新版。
- 可从版本历史中把任意有效版本设为最新版。
- 删除最新版时，系统自动选择最近的有效历史版本作为最新版。
- 删除版本只设置 `deletedAt`，不会删除 S3 对象。

## 章节目录

章节目录采用人工可修正方式，每行格式：

```text
章节名称|开始页|结束页
```

示例：

```text
产品零件清单|3|4
剥线|5|5
压接端子|6|6
装配线缆|7|7
灌胶|8|8
```

在右侧“目录”中点击章节即可跳到 PDF 对应页。目录页码不能小于 1，也不能超过已识别的 PDF 页数。

## PDF 预览

预览复用系统现有 `PdfViewer`，保留：

- 上一页 / 下一页。
- 页码输入跳转。
- 缩小 / 放大。
- 左旋 / 右旋 / 重置。
- 适应窗口 / 适宽 / 整页 / 原始大小。
- 全屏。
- 下载。
- Android WebView Promise / ArrayBuffer / PDF worker 兼容。

PDF 只渲染当前页，并预取相邻页元数据。切换版本时回到第 1 页。

## 图片集预览

图片集提供上一页、下一页、当前页 / 总页数、旋转、重置、适应窗口、原始大小、全屏和当前图片下载。底部文件条覆盖式显示，不压缩主预览区域。

## 关联连接器型号

在右侧“型号”页签点击“关联型号”：

1. 输入型号关键词搜索连接器参数。
2. 多选确认要绑定的参数记录。
3. 已关联记录会显示“已关联”，不能重复选择。
4. 需要解除时点击对应记录的“解除”。

系统不会自动绑定无法确认的型号。按型号关键词搜索只用于列出候选，最终由用户确认。

连接器参数表的“说明书 N”来自有效说明书绑定数量。`N=0` 时显示“暂无说明书”；`N>0` 时可进入说明书页并按型号筛选。

## 搜索

说明书页面支持：

- 标题。
- 制造商。
- 连接器系列。
- 文档编号。
- 摘要和关键词。
- 版本号。
- 关联连接器型号。
- PDF 正文。
- 说明书文件名。

顶部全局搜索新增“连接器组装说明书”和“说明书文件”分组。点击后进入：

```text
/connector-assembly-manuals?manualId=<manualId>&versionId=<versionId>&page=<pageNo>
```

本版本不引入 OCR。扫描型 PDF 需依靠标题、型号、制造商、版本和人工关键词检索。

## 下载

预览与下载都必须登录：

- 预览：`GET /api/connector-assembly-manual-assets/[assetId]/content`
- 下载：`GET /api/connector-assembly-manual-assets/[assetId]/download`

文件流接口使用 `Cache-Control: no-store` 和 `X-Content-Type-Options: nosniff`。数据库和 API 只保存 / 返回业务元数据，不返回对象存储密钥。

## 删除和恢复

- 删除说明书必须输入 `DELETE_MANUAL`。
- 删除版本必须输入 `DELETE_VERSION`。
- 文件移除会二次确认。
- 所有删除均为软删除。
- 回收站支持恢复说明书、版本和说明书文件。
- 不物理删除 S3 对象。
- 删除或恢复不会改变 `ConnectorParameter`。

## 操作日志

说明书新增、编辑、删除、恢复、版本上传、版本删除 / 恢复、设最新版、型号绑定 / 解除和下载都会写入 `operation_logs`。日志不记录密码、Token、数据库连接串或对象存储密钥。

## 数据和对象存储

元数据表：

- `ConnectorAssemblyManual`
- `ConnectorAssemblyManualVersion`
- `ConnectorAssemblyManualAsset`
- `ConnectorAssemblyManualBinding`
- `ConnectorAssemblyManualImportBatch`
- `ConnectorAssemblyManualImportItem`

对象路径：

```text
connector-assembly-manuals/{manualId}/{versionId}/{filename}
```

文件不会长期保存到容器本地。重新部署容器不会丢失 PostgreSQL 元数据或 S3 文件。

## 样本配置示例

用户样本 `ECTA24(32)芯插座（19年）.pdf` 建议配置：

- 名称：ECTA24(32)芯插座组装说明书
- 制造商：Amphenol AACS
- 版本：Rev 01
- 发布日期：2019-02-01
- 适用型号：ECTA24、ECTA32
- 章节：产品零件清单 3-4、剥线 5、压接端子 6、装配线缆 7、灌胶 8
- 关键词：ECTA24、ECTA32、CodeA、CodeB、CodeC、剥线、压接、装配、灌胶、VW60330、LV215、AXSON RE12602

该真实 PDF 未随当前任务作为可访问文件提供，因此仓库不包含样本，也没有伪造 8 页真实上传结论。部署前应按 QA 报告中的真实样本清单再验收一次。
