## 数据持久化说明

- 元数据存储在 Sealos PostgreSQL。
- 文件存储在 Sealos Object Storage。
- 所有登录账号共享同一套业务数据。
- 文件删除和工单删除均使用软删除，不物理删除对象存储文件。
- 重新部署应用容器不会丢失数据库记录或对象存储文件。
