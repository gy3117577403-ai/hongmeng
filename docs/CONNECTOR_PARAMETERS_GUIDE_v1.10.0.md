# v1.10.0 连接器参数资料使用指南

本文档适用于 v1.10.0-rc.1。文档不包含任何数据库密码、对象存储密钥、SESSION_SECRET 或账号密码。

## 什么是连接器参数资料

连接器参数资料是独立于生产工单的全局资料库，用于管理线束连接器剥皮与入长参数。

字段包括：
- 序号
- 型号
- 外剥皮mm
- 内剥皮mm
- 入长mm
- 备注
- 重点标记

参数字段允许为空，系统不会把空值自动填为 0。

## 如何进入

在顶部“资料库”菜单中选择“连接器参数资料”。

返回生产资料时，点击页面顶部“返回生产资料”。

## 如何新增参数

1. 点击“新增参数”。
2. 填写任意需要的字段。
3. 可勾选“重点标记”。
4. 点击保存。

说明：型号、外剥皮、内剥皮、入长和备注都允许为空，但整行不能完全为空。

## 如何编辑参数

在表格行右侧点击“编辑”，修改字段后保存。

型号允许重复，适合保留同型号不同批次、图号或备注条件下的参数。

## 如何删除 / 恢复

删除参数时会弹出二次确认。

删除为软删除，不物理删除数据库记录。需要恢复时点击“恢复删除”，在已删除参数列表中点击对应记录恢复。

## 如何搜索

顶部搜索框支持搜索：
- 型号
- 外剥皮参数
- 内剥皮参数
- 入长参数
- 备注

可以输入部分字符，例如：
- `116-1722`
- `信号针`
- `900V`
- `35`
- `参照图号`

输入会自动防抖查询，也可以按回车立即查询。

## 如何筛选缺失参数

筛选区支持：
- 全部
- 缺外剥皮
- 缺内剥皮
- 缺入长
- 任意缺失
- 重点标记

空字符串和空值都会被识别为缺失。

## 如何导入 Excel / CSV

1. 点击“导入 Excel / CSV”。
2. 选择“文件导入”。
3. 上传 `.csv`、`.xlsx` 或 `.xls` 文件。
4. 查看新增、跳过和失败明细。

支持中文表头：
- 序号
- 型号
- 外剥皮mm
- 内剥皮mm
- 入长mm
- 备注
- 重点

也支持英文表头：
- rowNo
- model
- outerPeelMm
- innerPeelMm
- insertionLengthMm
- remark
- isHighlighted

## 如何从 Excel 复制粘贴导入

1. 从 Excel 中复制包含表头的表格区域。
2. 打开“导入 Excel / CSV”。
3. 切换到“粘贴导入”。
4. 粘贴到文本框。
5. 点击“开始粘贴导入”。

系统会按 TSV / CSV 解析，空单元格保持为空。

## 如何导出 CSV

点击“导出 CSV”即可导出当前未删除参数数据。

导出使用中文表头，空值保持为空，重点字段导出为“是 / 否”。

## 如何上传原始资料

点击“上传原始资料”，可上传：
- PDF
- JPG / PNG
- CSV
- XLSX / XLS

原始资料文件保存到 S3 兼容对象存储，元数据保存到 PostgreSQL，不会长期存放在容器本地。

## 空值规则

- 外剥皮为空可以保存。
- 内剥皮为空可以保存。
- 入长为空可以保存。
- 备注为空可以保存。
- 型号为空可以保存。
- 整行完全为空不能保存。
- 导入时整行完全为空会跳过。
- UI 显示空值为浅灰“空”，不会显示为 0。

## 重点标记规则

重点标记用于把关键参数行显示为浅黄色背景。

导入时重点字段支持：
- 是
- 否
- true
- false
- 1
- 0

## 常见问题

**导入后空字段变成 0 吗？**  
不会。空字段会保持为空。

**型号可以重复吗？**  
可以。系统不对型号做唯一限制。

**原始资料会存到容器本地吗？**  
不会。文件会上传到对象存储。

**连接器参数和工单资料有关联吗？**  
没有强绑定。连接器参数是全局资料库，所有登录账号共享同一套数据。
