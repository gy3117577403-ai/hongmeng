# v1.10.0 连接器参数资料使用指南

本文档适用于 v1.10.0-rc.3。文档不包含任何数据库密码、对象存储密钥、SESSION_SECRET 或账号密码。

## 模块说明

连接器参数资料是独立于生产工单的全局资料库，用于管理线束连接器剥皮与入长参数。所有登录账号共享同一套数据，不做角色权限区分。

字段包括：序号、型号、外剥皮mm、内剥皮mm、入长mm、备注、重点标记。

参数字段允许为空，系统不会把空值自动填为 0。

## 进入方式

在顶部“资料库”菜单中选择“连接器参数资料”。返回生产资料时，点击页面顶部“返回生产资料”。

## 新增 / 编辑

点击“新增参数”可以创建一行参数。点击表格行右侧“编辑”可以打开弹窗修改字段。

型号允许重复，适合保留同型号不同批次、图号或备注条件下的参数。整行完全为空不能保存。

## 删除 / 恢复

删除参数会弹出二次确认。删除为软删除，不物理删除数据库记录。需要恢复时点击“恢复删除”，在已删除参数列表中点击对应记录恢复。

## 搜索和筛选

顶部搜索框支持搜索型号、外剥皮参数、内剥皮参数、入长参数和备注。搜索后表格标题会显示“查询结果 N 条”，命中的型号、参数和备注内容会用浅黄色背景标出。

筛选区支持：全部、缺外剥皮、缺内剥皮、缺入长、任意缺失、重点。空字符串和空值都会被识别为缺失。使用缺失筛选时，对应缺失单元格会显示浅橙底；在“全部”状态下空值保持干净空白。

## 导入预览流程

CSV、XLSX、XLS 和 Excel 粘贴导入都采用“导入预览 → 确认导入”流程。

1. 点击“导入 Excel / CSV”。
2. 选择文件，或切换到“粘贴导入”后粘贴表格。
3. 系统先解析并显示预览结果，不写入数据库。
4. 检查可导入、疑似重复、异常、跳过和重点行数量。
5. 选择重复行策略。
6. 点击“确认导入”后才写入数据库。

支持中文表头：序号、型号、外剥皮mm、内剥皮mm、入长mm、备注、重点。

支持英文表头：rowNo、model、outerPeelMm、innerPeelMm、insertionLengthMm、remark、isHighlighted。

模板外但有表头的非空列会自动合并到备注，例如真实 Excel 中的“波纹管”和“直径”列会保留为备注内容，避免补充资料在导入时丢失。

## 重复行规则

型号允许重复，但完全一样的数据默认不重复导入。

重复判断字段：
- model
- outerPeelMm
- innerPeelMm
- insertionLengthMm
- remark

比较前会 trim 空格，null 和空字符串视为一致。不比较 rowNo、isHighlighted、createdAt。

导入预览中，疑似重复行会标记为“疑似重复”。确认导入时默认跳过重复行；如果确实需要保留重复记录，可以选择“仍然导入重复行”。

## 重点行识别

最可靠方式是在表格中提供“重点”或 `isHighlighted` 列。

重点列支持：是、否、true、false、1、0、yes、no、Y、N。

如果没有重点列，系统会尝试识别 Excel 单元格黄色填充并自动标记重点。由于 `xlsx` 社区版对样式支持有限，黄色底色识别为 best-effort；无法识别时不会报错，只是不自动标重点。

## 批量操作

表格每行前有 checkbox。表头 checkbox 可全选当前页。

选中后会显示批量操作栏：
- 批量标记重点
- 批量取消重点
- 批量删除

批量删除为软删除，需要二次确认。批量操作会写入操作日志。

## 表格体验

顶部统计区采用一行紧凑统计条，表格区域优先占用页面高度。

表头保持 sticky。序号列和型号列在横向滚动时保持固定，型号列加粗显示，重点行显示浅黄色背景，当前选中行显示浅橙色背景。

操作列只显示“编辑 / 更多”。更多菜单包含标记重点 / 取消重点、复制整行参数和删除。

非重点行的重点列显示为空白；重点行显示黄色“重点”标签。

空值在表格中显示为空白，不显示 0、不显示“空”、不显示“-”。

## 复制整行参数

在每行“更多”菜单中点击“复制整行参数”，会复制当前行的型号、外剥皮、内剥皮、入长和备注。空值会保持为空，不会写成 0。

复制成功后页面提示“已复制参数”。复制操作会写入操作日志，日志只记录参数记录标识和基础摘要，不记录密码或密钥。

## 原始资料附件

可上传 PDF、JPG、PNG、CSV、XLSX、XLS 作为留档。文件保存到 S3 兼容对象存储，元数据保存到 PostgreSQL，不会长期存放在容器本地。

原始资料附件默认收起在右侧抽屉中。点击页面右侧“原始资料”按钮或统计条中的“附件”可打开抽屉。抽屉内支持上传、查看附件列表、下载和软删除。1024 宽度下抽屉覆盖式展开，不挤压表格。

## CSV 导出

点击“导出 CSV”即可导出当前未删除参数数据。导出使用中文表头，空值保持为空，重点字段导出为“是 / 否”。

## 样例 CSV

本地样例文件：

`docs/samples/connector-parameters-sample.csv`

样例包含空值、重复行、重点行和中文备注，用于本地导入预览与验收，不包含真实密码或密钥。

## 常见问题

**导入后空字段会变成 0 吗？**
不会。空字段会保持为空。

**型号可以重复吗？**
可以。系统不对型号做唯一限制，但完全相同的数据默认跳过。

**黄色底色识别一定准确吗？**
不保证。重点列识别最可靠，黄色底色识别为 best-effort。

**原始资料会存到容器本地吗？**
不会。文件会上传到对象存储。

**连接器参数和工单资料有关联吗？**
没有强绑定。连接器参数是全局资料库，所有登录账号共享同一套数据。
