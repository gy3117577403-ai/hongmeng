# 用户手册 v1.5.0

## 适用范围
本文档面向生产现场使用人员，适用于工单资料库 v1.5.0。系统采用账号登录方式，不区分角色权限，所有登录用户共享同一套业务数据。

## 登录
1. 打开生产地址。
2. 输入账号和密码。
3. 点击“登录”进入资料库首页。

如登录失败，请确认账号状态、密码是否正确，并联系管理员检查服务状态。

## 修改密码
1. 登录后点击顶部右侧用户入口。
2. 选择“修改密码”。
3. 输入当前密码、新密码和确认新密码。
4. 点击“保存”。
5. 修改成功后系统会提示重新登录。

新密码至少 6 位。请勿在公开环境记录或传播密码。

## 工单新增
1. 在左侧工单列表顶部点击“新建工单”。
2. 填写工单号、产品名称、阶段、优先级、状态、进度和备注。
3. 点击保存。

工单号必须唯一。新增后所有登录用户都可以看到该工单。

## 工单编辑
1. 在左侧选择目标工单。
2. 在右侧工单详情顶部点击“编辑工单”。
3. 修改产品名称、阶段、优先级、状态、进度或备注。
4. 保存后列表和详情会同步更新。

## 工单删除
1. 选择目标工单。
2. 点击“删除工单”。
3. 在确认弹窗中确认删除。

删除为软删除，默认列表不再显示该工单。删除工单不会物理删除对象存储中的文件。

## 工单搜索 / 筛选
- 搜索框支持按工单号或产品名称搜索。
- 筛选项包括全部、今日、本周、已完成、进行中。
- 筛选和搜索可以帮助生产现场快速定位工单。

## 资源分类使用
中间栏显示资料分类：
- 原图
- SOP指导书
- 成品图
- 辅料规格
- 注意事项
- 上传管理

原图、SOP指导书、成品图属于资料完整性判断中的必填分类。辅料规格、注意事项为可选分类。

## 批量上传 PDF / JPG / PNG
1. 选择工单。
2. 选择资料分类。
3. 点击上传按钮。
4. 选择一个或多个 PDF、JPG、PNG 文件。
5. 等待上传完成。

文件会保存到对象存储，元数据保存到 PostgreSQL。不要刷新页面或关闭浏览器，直到上传状态完成。

## 文件预览
1. 选择工单和分类。
2. 在文件列表中选择文件。
3. 右侧预览区会显示图片或 PDF 预览入口。

如果预览失败，请尝试刷新页面，或联系管理员检查对象存储访问状态。

## 文件下载
1. 选择文件。
2. 点击“下载当前”或文件操作中的下载按钮。
3. 浏览器会开始下载该文件。

## 下载全部 ZIP
1. 选择目标工单。
2. 点击右侧顶部“下载全部”。
3. 系统会打包当前工单下所有未删除文件。

ZIP 内部按资料分类组织目录。若当前工单没有文件，会提示暂无可下载文件。

## 文件编辑
1. 选择文件。
2. 点击“编辑文件”。
3. 修改显示名称或备注。
4. 保存后文件列表和详情会更新。

编辑文件信息不会覆盖原始对象存储文件。

## 上传管理
点击中间栏“上传管理”，可查看当前工单下所有未删除文件。支持按分类筛选，并可进行预览、下载、编辑和删除操作。

## 操作日志
点击顶部“操作日志”查看最近操作记录。日志包括上传、下载、删除、工单变更、修改密码、导入导出等操作摘要。

操作日志不会显示密码、对象存储密钥或数据库连接串。

## 系统设置
从顶部用户菜单或“批量导入”入口打开系统设置。系统设置包含：
- 系统基础信息
- 数据库状态
- 对象存储状态
- 上传大小限制
- 支持文件格式
- 数据导出
- 工单 CSV 导入
- 添加到桌面说明

## 工单 CSV 导入
1. 打开系统设置。
2. 下载 CSV 模板。
3. 按模板填写工单号、产品名称、阶段、优先级、状态、进度和备注。
4. 上传 CSV。
5. 查看导入结果汇总。

支持中文表头和英文表头。已存在且未删除的工单会更新；已软删除的同工单号记录会跳过。

## 数据导出
在系统设置中可导出：
- 工单 CSV
- 文件清单 CSV
- 操作日志 CSV
- 元数据 JSON

导出文件用于核对和交接，不包含密码、密钥或数据库连接串。

## 打印摘要
1. 选择目标工单。
2. 点击“打印摘要”。
3. 浏览器会打开打印流程。

打印内容包含工单摘要、分类文件数量和文件清单。打印样式会隐藏导航和按钮。

## 添加到桌面 / PWA 使用说明
1. 使用平板浏览器打开生产地址。
2. 打开浏览器菜单。
3. 选择“添加到桌面”或“安装应用”。
4. 添加后从桌面图标打开系统。

建议横屏使用。PWA 不会缓存业务 API、上传响应或签名下载链接。

## 常见问题
### 登录失败
确认账号和密码是否正确。若仍失败，请联系管理员检查服务状态。

### 上传失败
检查文件格式是否为 PDF、JPG 或 PNG，文件大小是否超出限制。若格式和大小正常，请联系管理员检查对象存储配置。

### 文件预览失败
尝试刷新页面或重新登录。如仍失败，请联系管理员检查对象存储公开访问端点。

### 下载全部失败
确认当前工单下是否存在未删除文件。如文件存在但仍失败，请联系管理员查看应用日志。

### CSV 导入失败
确认 CSV 使用 UTF-8 编码，工单号和产品名称已填写，进度在 0-100 之间。

### 数据刷新后仍旧
点击右侧“刷新”，或重新打开页面。若多人同时操作，以数据库中最新保存结果为准。
