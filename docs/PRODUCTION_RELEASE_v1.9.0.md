## 拍照上传说明

- 拍照入口位于右侧资料工具窗上传页、当前分类空状态和上传管理页。
- 浏览器优先请求后置摄像头，约束为 1920x1080 ideal。
- 拍照结果通过 canvas 生成 JPEG 文件，默认质量为 0.88。
- 照片文件复用现有上传 API，文件存入 Sealos Object Storage，元数据写入 PostgreSQL。
- 上传完成后刷新当前分类文件列表，并自动选中新上传图片。
- 关闭弹窗、重拍、上传完成或组件卸载时释放摄像头 media tracks。
- 不保存原始视频流，不记录图像内容。

## 语音输入说明

- 语音输入基于浏览器 `SpeechRecognition` / `webkitSpeechRecognition`。
- 默认语言为 `zh-CN`。
- 识别结果先写入当前输入框，用户可继续手动修改。
- 全局搜索语音输入会把识别结果写入搜索框，并触发已有搜索流程。
- 工单表单和文件信息表单不会因为语音输入而自动提交。
- 密码、账号密码、重置密码、修改密码、S3 设置和诊断信息不提供语音输入。
- 系统不上传音频到服务器，不保存录音，不把识别内容单独写入操作日志。

## 权限说明

- 拍照上传需要浏览器摄像头权限。
- 语音输入需要浏览器麦克风权限。
- 线上地址需要使用 HTTPS 才能稳定调用摄像头和麦克风能力。
- 用户拒绝权限时，系统只显示前端提示，不写入敏感日志。

## 已知兼容性说明

- 部分浏览器不支持 Web Speech API，此时语音输入按钮会显示不支持提示，用户可继续手动输入。
- 部分设备无法枚举多个摄像头，此时不显示有效的摄像头切换能力，但仍可使用默认摄像头或上传图片兜底。
- 拍照和语音能力受浏览器权限、设备硬件和 HTTPS 环境影响。

## 数据持久化说明

- 元数据存储在 Sealos PostgreSQL。
- 文件存储在 Sealos Object Storage。
- 所有登录账号共享同一套业务数据。
- 文件删除和工单删除均使用软删除，不物理删除对象存储文件。
- 重新部署应用容器不会丢失数据库记录或对象存储文件。
