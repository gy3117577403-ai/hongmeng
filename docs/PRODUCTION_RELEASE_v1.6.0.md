## 数据持久化说明

- 元数据存储在 Sealos PostgreSQL。
- 文件存储在 Sealos Object Storage。
- 所有登录账号共享同一套业务数据。
- 文件删除和工单删除均使用软删除，不物理删除对象存储文件。

## 回滚说明

- 如需回滚，通过 Sealos 将应用镜像改回上一稳定镜像 digest。
- 回滚前保留当前页面截图和应用日志。
- 回滚后验证 `/api/health`、登录、工单列表、上传、预览、下载和 ZIP 下载。
- 不要删除数据库。
- 不要删除对象存储 Bucket。
