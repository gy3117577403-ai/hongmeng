# v1.9.0 拍照上传与语音输入使用说明

本文档适用于 v1.9.0-rc.1。本说明不包含任何数据库密码、对象存储密钥、SESSION_SECRET 或账号密码。

## 如何拍照上传

1. 登录系统并打开目标工单。
2. 选择要上传到的资料分类。
3. 在右侧工具窗的“上传”页、当前分类空状态或上传管理页点击“拍照上传”。
4. 浏览器请求摄像头权限时选择允许。
5. 系统默认优先使用后置摄像头，并显示实时预览。
6. 点击“拍照”后进入照片预览。
7. 点击“确认上传”后，照片会使用现有上传接口保存到对象存储，元数据保存到 PostgreSQL。

## 如何重拍或连续拍摄

- 点击“重拍”会丢弃当前照片并重新打开摄像头。
- 点击“继续拍照”会暂存当前照片并返回实时预览，可以继续拍下一张。
- 点击“确认上传”会把已拍照片加入上传队列。
- 点击“取消”或关闭弹窗会释放摄像头。

## 摄像头权限问题

如果提示摄像头权限被拒绝，可以在浏览器站点权限中重新允许摄像头。

如果摄像头不可用，仍可点击“上传图片”选择 JPG / PNG 图片完成资料上传。

如果浏览器不支持直接调用摄像头，系统会显示拍照文件选择兜底入口。

## 如何使用语音输入

支持语音输入的输入框右侧会显示麦克风按钮。

1. 点击麦克风按钮。
2. 浏览器请求麦克风权限时选择允许。
3. 说出要输入的内容。
4. 识别内容会先写入当前输入框，用户可以继续手动修改。
5. 语音面板可执行停止、清空、应用到输入框和取消。

语音输入不会自动提交工单、文件信息或其他敏感表单。

## 哪些输入框支持语音

- 顶部全局搜索框。
- 新建工单：客户名称、工单号、产品名称、备注。
- 编辑工单：客户名称、产品名称、备注。
- 编辑文件信息：显示名称、备注。

密码、账号密码、重置密码、修改密码、S3 设置和诊断信息不提供语音输入。

## 浏览器不支持时怎么办

如果浏览器不支持 `SpeechRecognition` 或 `webkitSpeechRecognition`，麦克风按钮会显示不可用提示。此时继续手动输入即可，不影响任何业务流程。

## 隐私说明

- 系统不会保存录音。
- 系统不会把语音内容写入操作日志，除非用户最终保存到业务字段。
- 系统不会保存原始视频流。
- 摄像头只在拍照弹窗打开时使用，关闭、页面切换或上传完成后会释放。
- 线上使用拍照和语音能力需要 HTTPS。
