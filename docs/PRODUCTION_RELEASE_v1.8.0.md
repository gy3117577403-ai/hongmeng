## 功能清单

- PDF.js 自定义预览，支持翻页、缩放、适宽、整页和全屏。
- 图片完整预览，支持适应窗口、原始大小、缩放和全屏。
- 文件名安全解码，中文文件名正常显示。
- 同源文件内容流 API，用于登录态内读取文件内容。
- PDF worker API，用于 PDF.js worker 加载。
- 当前工单信息条显示客户、工单号、产品、状态、优先级、计划时间和资料完整性。
- 工单信息区去重，固定文件标题栏收口为悬浮文件胶囊。
- 右侧资料工具窗支持收起、展开和拖拽调整宽度。
- 右侧工具窗覆盖式展开，不挤压 PDF / 图片预览区。
- 缩略图默认收起，展开为透明浮动层。
- 主预览区优先布局，PDF / 图片占据更大可用空间。
- 顶部操作按钮收口，低频操作进入当前工单菜单或右侧工具窗。
- 1024x768 工作台优化，无明显横向溢出。
- 保留登录、修改密码、账号管理、工单管理、上传、下载、ZIP 下载、回收站、全局搜索、操作日志、系统设置、导入导出、PWA、PostgreSQL 和 S3 对象存储能力。

## PDF.js 预览说明

- PDF 预览不再依赖浏览器原生 PDF 工具栏。
- 前端使用 PDF.js canvas 渲染，默认适应宽度。
- 支持上一页、下一页、缩放、适宽、整页和全屏。
- PDF 控制条在预览区内浮动，不占用主布局高度。

## content API 说明

- API：`GET /api/resource-files/[id]/content`
- 用途：在登录态内读取当前文件内容，供 PDF.js 和图片预览使用。
- 安全说明：API 不返回对象存储密钥，不返回数据库连接串，不返回签名 URL 明细。

## PDF worker API 说明

- API：`GET /api/pdf-worker`
- 用途：为 PDF.js 提供 worker 脚本入口。
- 说明：该 API 仅服务前端 PDF 预览加载，不涉及业务数据写入。

## 数据持久化说明

- 元数据存储在 Sealos PostgreSQL。
- 文件存储在 Sealos Object Storage。
- 所有登录账号共享同一套业务数据。
- 文件删除和工单删除均使用软删除，不物理删除对象存储文件。
- 重新部署应用容器不会丢失数据库记录或对象存储文件。
