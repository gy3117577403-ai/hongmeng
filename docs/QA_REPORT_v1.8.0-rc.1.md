# QA Report v1.8.0-rc.1

## 版本信息
- 版本：v1.8.0-rc.1
- 状态：本地验证通过，等待统一部署
- 范围：资料预览与工作台排版优化
- 说明：当前报告不代表 Sealos 生产上线验收结果

## 功能清单
- PDF 预览改为 PDF.js canvas 渲染，避免浏览器原生黑色工具栏和侧边栏。
- 新增同源文件内容流接口，预览不暴露 S3 签名地址或对象存储 key。
- PDF 支持上一页、下一页、页码、缩放、适宽、整页和全屏预览。
- 图片预览支持适应窗口、原始大小、缩放和全屏预览。
- 文件名显示支持安全解码，显示名优先，原始文件名解码后回退。
- 资料分类侧栏、右侧资料信息 / 上传资料 / 文件操作区、顶部操作按钮和缩略图列表完成平板排版优化。
